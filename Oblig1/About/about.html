Use of HTML elements and CSS properties

One of our main goals from the start of the project, was to keep the website scalable to phone compatible, which is the reason for not using any exact pixel sizes for width or height. And also using the CSS property relative.

Div element was used for its dynamic and customizeble attributes, this was also a huge part of the scalability of the webpage.

We used alot of classes to help keep the webpages CSS not interfere with eachother, also som IDs were used for special circumstances where the one element need to change.

Most of the CSS was able to be in the stylesheet, but some had to be put into each pages own CSS element to make overrides where it was needed. Having most of the CSS in the stylesheet also helped keep the look and feel of all the pages the same theme.
